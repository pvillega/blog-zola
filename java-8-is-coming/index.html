Java 8 is coming

Posted by Pere Villega on 8 March 2014. Reading time: 1 minutes

It's March and Java 8 is coming soon. A delayed and very expected release of Java with plenty of new tools: lambdas, default methods, etc.

I've written about Java 8 and some of the new tools it provides in the Gumtree dev blog. There I list the changes along small code samples showing them in action.

As always, feedback via Twitter/Email is more than welcome.
